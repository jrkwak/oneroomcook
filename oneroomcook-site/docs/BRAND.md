# 원룸쿡 브랜드 가이드 v1.0

> 이 문서는 원룸쿡의 시각 아이덴티티와 보이스를 정의합니다.
> 협업 콘텐츠 제작, 외주 디자인, 그리고 브랜드 양도 시 이 문서가 기준입니다.

## 1. 브랜드 핵심

| 항목 | 정의 |
|---|---|
| 한 줄 정의 | 좁은 원룸, 최소한의 장비로 차린 근사한 한 끼 |
| 증명 명제 | "여섯 평에서도 식탁은 근사할 수 있다" |
| 3대 제약(세계관) | 1구 인덕션 · 조리대 60cm · 미니 오븐 |
| 슬로건 | 여섯 평에서도, 근사하게. |
| 타깃 ① | 20–30대 자취생·1인 가구 (레시피 독자) |
| 타깃 ② | 식당·브랜드 마케팅 담당자 (협업 파트너) |

## 2. 컬러 시스템

| 토큰 | HEX | 용도 |
|---|---|---|
| Cream `--c-cream` | `#FAF5EC` | 기본 배경 |
| Paper `--c-paper` | `#FFFDF8` | 카드 배경 |
| Ink `--c-ink` | `#2C2420` | 본문·제목 |
| Ink Soft `--c-ink-soft` | `#6E5F53` | 보조 텍스트 |
| **Terracotta** `--c-terracotta` | `#C05A3A` | 주 포인트 (CTA·링크) |
| Terra Deep `--c-terra-deep` | `#9A4326` | 포인트 hover |
| **Mustard** `--c-mustard` | `#D9A13F` | 보조 포인트·하이라이트 |
| Sage `--c-sage` | `#7C8B6C` | 카테고리 태그·차분한 강조 |
| Line `--c-line` | `#E9DFD1` | 구분선·테두리 |
| Wash `--c-wash` | `#F3EADD` | 섹션 배경 |

규칙: 테라코타는 화면당 1–2곳(CTA)에만. 배경은 항상 크림 계열 — 순백(#FFF)은 쓰지 않습니다.

## 3. 타이포그래피

| 역할 | 서체 | 비고 |
|---|---|---|
| 제목·인용 | **Noto Serif KR** (400/600/700) | 에디토리얼 무드의 핵심 |
| 본문·UI | **Pretendard Variable** | 가독성·모바일 최적 |

- 제목은 세리프, 본문은 산세리프 — 절대 섞어 쓰지 않습니다.
- 한글 줄바꿈: `word-break: keep-all` 전제 하에 카피 작성.
- 영문 키커(kicker)는 대문자 + letter-spacing 0.18em.

## 4. 로고

- 마크: `assets/img/logo-mark.svg` — 원룸(사각 프레임) 안의 접시와 김. 최소 사용 크기 24px.
- 워드마크: 텍스트 "원룸쿡" — '쿡'만 테라코타. HTML에서는 `.brand-name` 컴포넌트 사용.
- 금지: 마크 회전·색 변경, 저채도 배경 외 사용, 그림자 추가.

## 5. 사진·일러스트 톤

- 사진: 자연광, 원룸 주방의 실제 맥락(좁은 조리대, 1구 인덕션)이 보이게. 과한 보정·스튜디오 톤 금지.
- 일러스트(임시 자산): 브랜드 팔레트의 플랫 스타일. `scripts/build.py`가 생성.
- 모든 이미지는 4:3(카드) 또는 1:1(피드) 비율.

## 6. 보이스 & 톤

- **진솔하게**: 실패 지점("여기서 태우기 쉽습니다")을 숨기지 않는다.
- **현실적으로**: "생크림 대신 슬라이스 치즈" — 자취방 냉장고 기준으로 말한다.
- **단정하게**: 과장 광고형 표현("인생 레시피!", "무조건 성공!") 금지.
- 광고주 대상 문서(미디어킷)에서는 수치 기반으로, 추정치는 추정치라고 명시한다.
